Security posture

Trust is designed into each Companion stop.

The site stays static, the installer is hash-verifiable, credentials stay local, and the gateway defaults to a loopback Windows boundary.

Website boundary

GitHub Pages serves static HTML and CSS only.

CSP

Meta CSP

Every page blocks scripts, objects, forms, and foreign base URLs with a meta Content-Security-Policy.

JS

No runtime JavaScript

The journey uses links, anchors, tables, and CSS transitions only. There are no script tags or inline handlers.

CSS

No inline style

Presentation lives in assets/styles.css so the audit surface is small and reviewable.

GitHub Pages does not expose custom HTTP security headers for this static repository, so the site uses conservative markup, a meta CSP, no forms, no object embeds, and no third-party runtime assets.

Installer integrity

Verify the EXE before you run it.

The published installer SHA-256 should match this value:

a1a1154f8816c3fa26100224182df9842c00777026cda1b6afff248d501781ce

Get-FileHash .\OpenClawWindowsNativeSetup.exe -Algorithm SHA256
Get-Content .\checksums.sha256

Open checksums.sha256

Secrets

The public package never ships real credentials.

Telegram bot token

Entered by the user during setup and stored under %USERPROFILE%\.openclaw\credentials as a token file.

Gateway token

Generated or selected locally by OpenClaw and never published in Pages, release assets, or screenshots.

Provider credentials

Model API keys or OAuth information are handled by official OpenClaw onboarding, not bundled in this kit.

Local Windows boundary

Gateway defaults stay on the participant PC.

The installer recommends local gateway mode, loopback bind, token auth, and port 18789. Desktop launchers operate from the user's Windows profile.

During classes or screen shares, do not display token files, pairing codes, API keys, OAuth sessions, or install logs that may include local paths.

Supply chain

Pin the upstream OpenClaw ref when reproducibility matters.

The default installer uses the OpenClaw upstream main ref. For a workshop, lab, or incident review, use a reviewed tag or commit with -RepoRef.

powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\Install-OpenClawWindowsNative.ps1 -RepoRef <tag-or-commit>
