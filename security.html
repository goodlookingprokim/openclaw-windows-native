English mode

Secrets stay local; public files stay verifiable

Pages use HTML and CSS only, with scripts blocked. Release assets do not include Telegram tokens, Gateway tokens, API keys, pairing codes, local state, or logs.

  1. Compare SHA-256 before running the installer.
  2. The Gateway defaults to loopback 127.0.0.1:18789.
  3. Telegram uses a user-local token file.
  4. Dry-run artifacts are redacted and simulated.
