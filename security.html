Local boundary

Gateway는 기본적으로 내 PC 안에서만 열립니다.

English mode

The rule is simple: secrets stay local.

The public site ships static HTML and CSS only. The installer package does not include Telegram tokens, Gateway tokens, API keys, pairing codes, local state, or logs.

Before live Telegram setup, dry-run creates a redacted validation artifact. Live send/receive still requires your own BotFather token and pairing approval.
