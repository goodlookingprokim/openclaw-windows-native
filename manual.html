02 Guided Setup

Let the installer prepare the Windows-native workspace.

A

Prerequisites

Windows 10/11, internet access, and PowerShell or CMD. The installer checks Git, Node.js LTS, and pnpm.

B

Native install

The kit clones OpenClaw into %USERPROFILE%\openclaw-src, installs dependencies, builds, and creates desktop launchers.

C

Safe defaults

Gateway mode is local, bind is loopback, auth is token, and the default port is 18789.

powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\Install-OpenClawWindowsNative.ps1 -RepoRef main

Generated paths

Know where the Companion puts things.

PathPurpose
%USERPROFILE%\openclaw-srcOpenClaw source checkout, build output, and CLI working directory.
%USERPROFILE%\.openclawConfig, logs, sessions, and user-local credentials.
%USERPROFILE%\.openclaw\credentials\telegram-bot-token.txtTelegram bot token file created on the participant PC only.
%USERPROFILE%\Desktop\OpenClawStart, status, stop, dashboard, pairing, update, and verify helpers.

03 Telegram Connect

Create a bot, store the token locally, then approve pairing.

BOT

BotFather

In Telegram, message @BotFather, run /newbot, name the bot, and copy the bot token when the installer asks for it.

FILE

Token file

The token is written to a user-local file path and registered with OpenClaw as channels.telegram.tokenFile.

cd $env:USERPROFILE\openclaw-src
pnpm.cmd openclaw plugins enable telegram
pnpm.cmd openclaw channels add --channel telegram --token-file "$env:USERPROFILE\.openclaw\credentials\telegram-bot-token.txt"

04 Verify Conversation

Prove the gateway and Telegram channel are alive.

Send a message to your Telegram bot. If OpenClaw requires pairing, list pending codes and approve the code shown for your conversation.

cd $env:USERPROFILE\openclaw-src
pnpm.cmd openclaw pairing list telegram
pnpm.cmd openclaw pairing approve telegram <PAIRING_CODE>
pnpm.cmd openclaw gateway health
pnpm.cmd openclaw channels status --probe

You can also use OpenClaw_02_Status.cmd and OpenClaw_05_Approve_Telegram_Pairing.cmd from the desktop OpenClaw folder.

05 Repair/Update

When something drifts, use the desktop helpers before guessing.

Gateway stopped

Run OpenClaw_01_Start_Gateway.cmd, then OpenClaw_02_Status.cmd. Check that port 18789 is listening.

Dashboard token issue

Open the dashboard through OpenClaw_04_Open_Dashboard.cmd so the current gateway token is applied.

Telegram silent

Run channels status --probe, inspect pending pairings, then approve the latest pairing code.

Update/rebuild

Run OpenClaw_06_Update.cmd to pull the selected repo ref, rebuild, re-enable Telegram, and restart the gateway.

Full verifier

Run the verifier after a repair to check dependencies, build output, config, gateway health, channel probe, and listener state.

Clean uninstall

Use the included uninstall script only when you intentionally want to remove scheduled startup and local helper files.

powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\Verify-OpenClawWindowsNative.ps1
