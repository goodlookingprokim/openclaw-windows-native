Bilingual companion

Windows에서 OpenClaw를 쉽게 설치하고 Telegram으로 확인합니다.

한국어와 English를 탭으로 전환합니다. 어려운 절차를 작은 행동으로 나눕니다.

한국어 mode

한 번에 하나씩 진행하면 됩니다.

목표는 단순합니다. Windows에서 설치 파일을 실행하고, 실제 Telegram 토큰을 넣기 전에 dry-run으로 흐름을 확인한 뒤, 본인 봇과 첫 대화를 검증합니다.

  1. 설치 파일을 받습니다.
  2. SHA-256 지문을 확인합니다.
  3. Windows 네이티브 PowerShell/CMD에서 설치합니다.
  4. dry-run으로 민감정보 없이 연습합니다.
  5. BotFather 토큰을 로컬 파일에 저장하고 pairing을 승인합니다.

English mode

Use one small step at a time.

The goal is simple: install OpenClaw on native Windows, practice the Telegram path with dry-run, then connect your own bot and verify the first conversation.

  1. Download the setup file.
  2. Compare the SHA-256 fingerprint.
  3. Run setup from native Windows PowerShell/CMD.
  4. Practice with dry-run before real secrets.
  5. Store the BotFather token in a local file and approve pairing.
