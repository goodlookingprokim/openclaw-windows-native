OpenClaw Windows Native Kit

WSL 없이 Windows에서 OpenClaw Gateway를 설치하고 Telegram으로 독립형 에이전트를 실험할 수 있도록 만든 강의/연구용 배포 키트입니다.

설치 파일 다운로드 설치 매뉴얼 보기

SHA-256: f00a55ddaa0f870d98fef4e98c82bb2cc454564b632df8d10a23ff4a599a3d0b

무엇을 해결하나요?

OpenClaw를 처음 접하는 Windows 사용자도 Git, Node.js, pnpm, OpenClaw 빌드, Gateway 실행, Telegram bot token 등록, pairing 승인을 한 흐름으로 진행할 수 있게 합니다. 설치 파일에는 토큰이나 API key가 들어 있지 않고, 사용자가 직접 선택해 입력합니다.

설치 흐름

1

설치 파일 실행

Windows에서 EXE를 실행하면 PowerShell 기반 설치기가 시작됩니다.

2

필수 도구 확인

Git, Node.js LTS, pnpm을 확인하고 가능한 경우 winget으로 설치합니다.

3

OpenClaw 온보딩

모델/provider 인증과 Gateway 설정은 공식 OpenClaw 온보딩에서 직접 선택합니다.

4

Telegram 연결

BotFather 토큰을 로컬 token file로 저장하고 pairing code를 승인합니다.

다운로드

가장 쉬운 방법은 설치 파일 하나를 내려받아 실행하는 것입니다. 학교나 회사 PC에서 EXE 실행이 제한되면 PowerShell 스크립트를 직접 실행할 수 있습니다.

OpenClawWindowsNativeSetup.exe 권장 설치 파일 Install-OpenClawWindowsNative.ps1 PowerShell 설치기 checksums.sha256 무결성 확인

문서와 운영

사용자 설치 매뉴얼 기술 스펙 OpenClaw CLI 문서 OpenClaw Telegram 문서
