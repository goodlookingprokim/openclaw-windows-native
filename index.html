한국어 mode

처음 설치하는 사람도 한 단계씩 따라갈 수 있습니다.

목표: 참석자가 설치 파일을 실행하고, 본인 Telegram 봇 토큰과 모델 인증을 직접 입력해 Windows 네이티브 환경에서 OpenClaw를 사용한다.

설치는 다운로드, 확인, 실행, dry-run, Telegram 연결, 대화 확인으로 나누면 쉽습니다.

  1. 설치 파일을 받습니다.
  2. SHA-256 지문을 확인합니다.
  3. Telegram dry-run으로 먼저 연습합니다.
  4. 실제 토큰은 로컬 파일에만 저장합니다.

English mode

Follow one small step at a time.

OpenClaw setup becomes easier when it is split into plain actions: download, check, install, practice safely, connect Telegram, and verify the first conversation.

  1. Download the setup file.
  2. Compare the SHA-256 value.
  3. Run Telegram dry-run before real credentials.
  4. Store the BotFather token only in a local user file.
  5. Approve pairing and verify the Telegram channel.
