The route

A companion path, not a pile of files.

OpenClaw setup has several trust-sensitive moments: an EXE download, local build tools, Telegram bot credentials, pairing approval, and gateway recovery. The redesigned Pages flow keeps those moments in one visible sequence.

Every page points back to the same journey so a classroom participant, maintainer, or security reviewer can land anywhere and still know the next safe action.

Companion map

Download -> Guided Setup -> Telegram Connect -> Verify Conversation -> Repair/Update

01

Download

Get the public kit and compare its SHA-256 before running it.

 02

Guided Setup

Install Git, Node.js LTS, pnpm, clone OpenClaw, build, and create desktop launchers.

 03

Telegram Connect

Use BotFather, store the bot token as a local file, and enable the Telegram channel.

 04

Verify Conversation

Dry-run safely first, then start the gateway, approve pairing, and confirm the first Telegram probe.

 05

Repair/Update

Use desktop helpers and the verifier to recover, rebuild, or update OpenClaw.
