OpenClaw Windows Companion

Windows에서 OpenClaw를 설치하고 Telegram 대화 준비까지 확인합니다.

WSL 없이 설치하고, 실제 비밀 값은 로컬에만 두며, dry-run으로 먼저 안전하게 연습합니다.

WSL 불필요한국어 단일 안내토큰 비공개Dry-run 먼저

처음 사용자를 위한 흐름

설치부터 Telegram 대화 준비까지 한 번에 따라갑니다.

OpenClaw를 Windows 네이티브 환경에서 설치하고, 실제 Telegram 토큰을 넣기 전에 dry-run으로 흐름을 확인한 뒤, 본인 봇과 첫 대화를 검증합니다. 모든 안내는 한국어 단일 화면으로 정리했습니다.

  1. 설치 파일을 받고 SHA-256 지문을 확인합니다.
  2. Windows PowerShell 또는 CMD에서 설치를 실행합니다.
  3. 민감정보 없이 dry-run으로 먼저 연습합니다.
  4. BotFather 토큰은 로컬 tokenFile에만 저장합니다.
  5. pairing과 channel probe로 대화 준비 상태를 확인합니다.

진행 경로

다운로드 → 설치 → Dry-run → Telegram → 검증
